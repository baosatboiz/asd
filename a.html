06 · Cart Service :8005
Client / Gateway / customer-service
      |
      v
CartViewSet  (DRF ModelViewSet + custom actions)
      |
      +——> GET    /carts/                  ———> list carts
      +——> POST   /carts/                  ———> create cart          <—— called by customer-service on register
      +——> GET    /carts/{id}/             ———> retrieve cart
      +——> PATCH  /carts/{id}/             ———> update cart
      +——> DELETE /carts/{id}/             ———> delete cart
      |
      +——> POST   /carts/{id}/add-item/    ———> add or increment item
      |               |
      |               +——> GET /books/{id}/ ——> book-service:8004    (validate book exists)
      |               |         |
      |               |         +——> book not found: return 404
      |               |         +——> book found: continue
      |               |
      |               +——> CartItem exists? ——> YES: quantity++
      |               +——————————————————————> NO:  create CartItem
      |
      +——> POST   /carts/{id}/clear-items/ ———> delete all CartItems  <—— called by gateway after checkout
      |
      v
CartSerializer + CartItemSerializer
      |
      v
Cart DB (SQLite)          CartItem DB (SQLite)
  customer_id (unique)        cart (FK → Cart)
  is_active                   book_id
  created_at                  quantity
                              unique_together (cart, book_id)
07 · Order Service :8006
Client / Gateway
      |
      v
OrderViewSet  (DRF ModelViewSet)
      |
      +——> GET    /orders/       ———> list orders    <—— called by comment-rate, recommender-ai, gateway
      +——> GET    /orders/{id}/  ———> retrieve order
      +——> PATCH  /orders/{id}/  ———> update order   <—— called by gateway staff mark-shipped
      +——> DELETE /orders/{id}/  ———> delete order
      |
      +——> POST   /orders/       ———> create order
      |               |
      |               +——> Order DB (write — status: "pending")
      |               |
      |               +——> POST /payments/ ——> pay-service:8008
      |               |         |
      |               |         +——> Payment DB (write — status: "paid")
      |               |         +——> return payment status
      |               |
      |               +——> POST /shipments/ ——> ship-service:8007
      |               |         |
      |               |         +——> Shipment DB (write — status: "processing")
      |               |         +——> return shipment status
      |               |
      |               +——> payment success? ——> update Order.status = "paid"
      |               +——> return order + payment_status + shipping_status
      |               |
      |               NOTE: shipping_address encodes book IDs as  |books:1,2,3
      |
      v
OrderSerializer
      |
      v
Order DB (SQLite)
      customer_id, total_price, status (pending|paid|shipped|completed)
      shipping_address  ⚠ contains |books:1,2,3 suffix
      created_at

      |               |
      |               +——> GET /orders/ ——> order-service:8006
      |               |         |
      |               |         +——> fetch all orders for this customer
      |               |         +——> extract book IDs from order.shipping_address  |books:1,2,3
      |               |              (fallback: parse ordered_book_ids field if present)
      |               |
      |               +——> book_id in purchased books?
      |               |         |
      |               |         +——> NO:  return 403 Forbidden
      |               |         +——> YES: save CommentRate to DB
      |
      v

      v

11 · Recommender-AI Service :8010
Client / Gateway
      |
      v
RecommendationViewSet  (custom list())
      |
      +——> GET /recommendations/   ———> dynamic scored list  (no precomputed data)
      |               |
      |               +——> GET /orders/ ——————————> order-service:8006
      |               |         |
      |               |         +——> fetch all orders
      |               |         +——> parse purchased book IDs from shipping_address  |books:...
      |               |         +——> count how many times each book was ordered
      |               |
      |               +——> GET /comments-rates/ ——> comment-rate:8009
      |               |         |
      |               |         +——> fetch all ratings
      |               |         +——> filter: rating > 4 only
      |               |         +——> compute avg high rating per book
      |               |
      |               +——> score each book:
      |               |         score = (order_count × 10) + (avg_high_rating × 20)
      |               |
      |               +——> read X-Actor-Id header (current user)
      |               +——> remove books already ordered by this user  (personalization)
      |               +——> return top 20 with explanation text
      |
      v
Recommendation DB (SQLite)
      (stored model exists but list() is fully dynamic — no DB writes during recommendations)
